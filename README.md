# 幻晶モンスターズ v1.3 戦闘コマンド強化版

スマホ縦画面向けのオリジナル・モンスター育成RPGプロトタイプです。

## 公開用ファイル

GitHub Pagesでは `index.html` が表示されます。
このZIPを解凍して、中身をリポジトリ直下にアップロードしてください。

## デバッグ確認

静的構文チェックと、Node上の疑似DOMで以下の流れを確認しています。

- タイトル開始
- イベントスキップ
- 探索開始
- 戦闘開始
- 命令 → 味方選択 → スキル選択 → 対象選択
- 道具 → 回復アイテム → 対象選択
- 作戦 → 味方選択 → 方針保存
- スカウト成功 → 名前入力 → 仲間追加
- モンスター詳細タブ表示

## 注意

既存作品の画像、ロゴ、キャラクター、UI素材は使用していません。
CSSとHTMLで構成したオリジナルのプロトタイプです。

## v1.8 戦闘システム修正版

修正内容:

- 「戦う」を押したとき、パーティメンバー全員が1ターン内で行動するように変更
- 命令なしの場合:
  - パーティメンバー全員が作戦に応じてオート行動
- 「命令」を使う場合:
  - パーティメンバー全員の行動を選択
  - 全員分の命令がそろったあと「戦う」で命令どおりに行動開始
- スカウト仕様を変更
  - 1戦闘でスカウト対象は1体のみ
  - スカウト成功時は敵が残っていても戦闘終了
  - スカウトしたモンスター分の経験値/Gは通常どおり獲得
  - スカウト成功後は名前入力と加入先選択へ進む

メインファイル:

```text

```
